PROMPT PARA GERAR SCRIPT PYTHON DE FEATURES DE SIMILARIDADE ENTRE MARCAS

Crie um script completo em Python para gerar um dataset de features de
similaridade nominativa entre pares de marcas, com foco em posterior
treinamento supervisionado de um modelo de classificação. O script não
deve atribuir rótulo, apenas gerar todas as features e salvar em arquivo
para posterior rotulagem manual.

OBJETIVO Ler um arquivo contendo pares de nomes de marcas, calcular um
conjunto robusto de features de similaridade e gerar um arquivo tabular
contendo: - nomes originais das marcas - versões normalizadas - features
numéricas e booleanas - colunas auxiliares para rotulagem humana

ENTRADA Arquivo CSV ou XLSX contendo ao menos: marca_a marca_b

O caminho do arquivo deve ser passado por argumento de linha de comando.

SAÍDA Gerar CSV (e opcionalmente XLSX) contendo:

Colunas originais: marca_a_original marca_b_original

Normalizações: marca_a_lower marca_b_lower marca_a_sem_acentos
marca_b_sem_acentos marca_a_alnum marca_b_alnum marca_a_alpha_only
marca_b_alpha_only marca_a_sem_espacos marca_b_sem_espacos
marca_a_repeat_compressed marca_b_repeat_compressed marca_a_fonetica
marca_b_fonetica

Apoio à rotulagem: tokens_a tokens_b tokens_a_sorted tokens_b_sorted
split_candidates_a split_candidates_b

NORMALIZAÇÃO Criar funções para: - lowercase - remoção de acentos -
remoção de pontuação - apenas alfanumérico - apenas letras - remoção de
espaços - compressão de repetição de letras - tokenização - tokens
ordenados

Tratar separadores: - hífen - underscore - barra - ponto - vírgula -
apóstrofo

FONÉTICA Implementar função fonética heurística para português:

Regras mínimas: ph → f y → i w → v ou u remover h mudo comprimir
consoantes repetidas

aproximações: c k q s z x ç ss g j

Remover acentos antes das regras.

TOKENIZAÇÃO Gerar tokens por espaço e separadores.

Implementar tentativa de decomposição heurística para casos como:
bionovo ↔ bio novo novobio ↔ novo bio autodias ↔ auto dias

Permitir lista opcional de radicais: bio novo auto max plus prime tech
smart flex vida car
